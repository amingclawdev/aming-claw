# Archived Aming Claw Skills

These files preserve the pre-onboard skill documents for provenance. They are
not active Codex or Claude plugin entrypoints.

Use `skills/aming-claw-onboard/SKILL.md` as the single active entrypoint. From
there, follow MCP `onboard_route_guide` and its returned role, token,
next-action, interface, and contract-chain indexes. Use the HTTP endpoint only
when the MCP tool is unavailable.

The machine-readable migration map is `Archive/skills/index.json`.
